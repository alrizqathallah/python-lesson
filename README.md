# Materi Python

## Tentang Python

**Python** adalah bahasa pemrograman yang sangat populer didunia, pertama kali dikembangkan oleh Guido Van Rossum pada 1991.

### Penggunaan Python

- Membangun aplikasi web (server-side)
- Pengembangan Perangkat Lunakn (Software)
- Perhitungan Matematika
- dan Sistem Skrip (Scripting)

### Apa yang dapat dilakukan dengan Python

- Python dapat digunakan untuk mengembangkan server pada aplikasi web.
- Python juga dapat digunakna bersamaan dengan perangkat lunak guna membangun sebuah alur kerja.
- Juga dapat dikoneksikan dengan Database, guna membaca data dan memanipulasi data.
- Melakukan perhitungan matematikan yang kompleks dan kebutuhan _Big Data_.
- Bisa pula dimaksimalkan untuk pengembangan prototip perangkat lunak ata _production ready_.

## Kenapa Python?

- Python dapat dijalankan disemua sistem operasi (Windows, Linux, Mac, Pi, dll.)
- Memiliki sintaks dasar yang simpel, sehingga ramah untuk pemula.
- Menggunakan sistem _interpreter_, sehingga kode bisa langsung dijalankan setelah selesai ditulis, tanpa perlu dikompilasi manual.
- Dapat menjalankan bermacam paradigma pemrograman, _Procedural_, _Object-Oriented_ dan _Functional_.

### Perlu diketahui

- Versi Python yang umum digunakan saat ini (2026) adalah Python 3 (versi 3.xxx).

### Perbandingan Sintaks Python dengan lainnya

- Python didesain dengan tingkat _Readability_ yang tinggi, sehingga lebih mudah dipahami.
- Dengan bahasa inggris yang dipengaruhi unsur Matematika.
- Python tidak menggunakan Semicolon **;** untuk menutup sebuah _statement_ kode.
- Tidak menggunakan _parentheses_ atau _curly bracket_ untuk membangun blok kode.
- Menggunakan _space-indentation_ untuk menulisan blok kode yang akan dieksekusi.
