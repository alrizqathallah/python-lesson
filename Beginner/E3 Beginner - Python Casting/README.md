# Python Casting

## Tipe Variabe Spesifik

Terkadang kita akan sering menentukan tipe data tertentu dalam sebuah variable.

Python adalah bahasa pemrograman yang berorientasi objek.

Dalam praktiknya sering menggunakan `class` dalam mendefinisikan sebuah data, termasuk tipe data primitif.

Seperti `int()`, `float()`, `str()`

contoh:

```Python
# untuk int()
x = int(5) # x sama degan 5
y = int(3.7) # y sama dengan 3
z = int("2") # z sama dengan 2 dalam integer bukan string

# untuk float
x = float(1)     # x will be 1.0
y = float(2.8)   # y will be 2.8
z = float("3")   # z will be 3.0
w = float("4.2") # w will be 4.2

# untuk str
x = str("s1") # x will be 's1'
y = str(2)    # y will be '2'
z = str(3.0)  # z will be '3.0'
```
