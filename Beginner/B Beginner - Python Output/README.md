# Python Output

## Print Teks

`print()` statement dapat digunakan untuk mencetak output.

contoh:

```Python
print('Hello World')
```

statement tersebut dapat dibuat sebanyak mungkin yang diinginkan.

contoh:

```Python
print('Hello World')
print('Hello World')
print('Hello World')
print('Hello World')
print('Hello World')
```

Teks yang dicetak menggunakan print statement harus dibungkus dengan `'` atau `"`, sebagai penanda string.

Jika terlupa untuk membungkusnya menggunakan _quotation-mark_ maka akan memunculkan eror sebelum kode dieksekusi, pada sintaks.

Secara default, `print()` statement membuat baris baru ketika output ditampilkan. Namun dapat diatasi dengan `end` statement untuk mencetak dalam baris yang sama atau tidak membuat baris baru.

```Python
print('Hello World', end=' ')
print('print in the same line')
```

## Print Bilangan / Angka

Sama dengan mencetak teks, namun tidak dibutuhkan _quatation-mark_ untuk mencetak angka.

contoh:

```Python
print(1)
print(10)
print(100)
print(1000)
print(10000)
print(3.14)
```

dengan statement print, juga dapat melakukan operasi secara langsung.

contoh:

```Python
print(1 + 4)
print(1 * 4)
print(5 - 4)
print(8 / 4)
```

## Print Teks dan Bilangan secara bersamaan

contoh:

```Python
print('I am', 28, 'years old')
```
