# Python Comment

Comment atau komentar digunakan untuk menjelaskan kode python yang dituliskan.

Komentar menambah keterbacaan dan pengertian dari sebuah kode yang ditulis.

Komentar juga memberikan penjelasan detail, sebelum sebuah kode itu diuji atau dijalankan.

## Membuat komentar

Dalam python, komentar dibuat dengan menggunakna simbol `#` _pawn_, yang diikuti isi dari komentar.

contoh:

```Python
# Ini adalah Print Statement
print('Hello World')
```

Komentar juga dapat ditempatkan setelah sebuah statement.

contoh:

```Python
print('Hello World') # Ini adalah Print Statement
```

Komentar menggunakan _pawn_ `#`, hanya dapat mencakup satu baris kode saja.

Untuk komentar yang mencakup lebih dari satu baris kode _multiline-comment_, dapat menggunakan simbol `"""` yang ditutup juga dengan simbol `"""`.

contoh:

```Python
""" ini
adalah
komentar
multi
baris
"""
```

Secara kerja, komentar akan diabaikan oleh interpreter, dan tidak akan muncul pada output.

Komentar hanya dapat dilihat langsung pada sintaks kode yang dijalankan.
